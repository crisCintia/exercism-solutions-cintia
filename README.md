# My Exercism Python Journey 🚀

This repository contains my solutions to the challenges in the **Python** track on [Exercism](https://exercism.org/). The focus of this study is to strengthen the fundamentals of programming logic for application in **Data Analysis and Artificial Intelligence**.

## 📌 About the Project

Exercism provides practical exercises based on fundamental concepts. Each exercise is validated by unit tests, ensuring that the logic is correct and follows the language's best practices (PEP 8 and PEP 257).

## 🛠️ Technologies Used

* **Python 3.x**
* **Unit Tests** (Pytest)
* **Best Practices:** Use of Constants, Docstrings, and Clean Code.

## 📖 Completed Exercises

### 1. Guido's Gorgeous Lasagna 🍝
* **Concepts learned:** Variables, Constants, Functions, and Docstrings.

* **Summary:** Implementation of a system to calculate preparation and cooking times for a lasagna, avoiding the use of *Magic Numbers*.
